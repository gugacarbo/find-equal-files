# find-equal-files

## Install

```bash
$ npm install --global find-equal-files
```

## CLI

```
$ find-equal-files --help

  Usage
    $ find-equal-files

  Options
      --path  /path/to/directory

  Examples
  # In current directory
    $ find-equal-files

  # With relative path
    $ find-equal-files --path=/path/to/directory

  # With absolute path
    $ find-equal-files --path=C:\path\to\directory

```
